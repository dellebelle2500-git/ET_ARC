# ET_ARC — Figure code

Reproducible figure-generation code for the study of genetic predictors of
response to a yeast-derived aldehyde-reducing compound (ARC) in patients with
essential tremor (ET).

Each script is self-contained: it reads the de-identified data in [`data/`](data/),
regenerates one manuscript figure as a vector **PDF** (plus a raster **PNG**
preview), and writes it to `output/`. Figures are laid out at *Nature
Communications* dimensions (180 mm double-column width) with fonts embedded as
editable TrueType.

## Repository structure

```
ET_ARC/
├── figures/                 # main manuscript figures (canonical, self-contained)
│   ├── figure1.py           # Fig. 1  efficacy + data-driven subgrouping (6 panels)
│   ├── figure2.py           # Fig. 2  dual-framework locus discovery (3 panels)
│   ├── figure3.py           # Fig. 3  two-axis pharmacogenomic model (2 panels)
│   └── figureS3.py          # Fig. S3 per-locus dosage on PCA embedding (4×4)
├── panels/                  # standalone single-panel scripts (building blocks)
│   ├── make_timecourse.py         make_dosage_boxstrip.py   make_gene_score.py
│   ├── make_ward_boxstrip.py      make_freq_barplot.py      make_crst_gene_score.py
│   ├── make_pca_basic.py          make_volcano.py           make_twoaxis_scatter.py
│   ├── make_dendrogram.py         make_pca_overlay.py
├── data/                    # de-identified inputs (see data/README.md)
│   ├── patients_clustering.csv    snp_dosage_matrix_final.csv
│   ├── CRST.csv                   QUEST.csv
│   └── stats/                     per-SNP association statistics
├── output/                  # generated figures (git-ignored)
├── requirements.txt
└── README.md
```

`figures/` contains the canonical scripts that reproduce the published figures;
each is fully self-contained. `panels/` holds the standalone per-panel scripts
used during development — useful if you want an individual panel in isolation
(e.g. for re-assembly in a vector editor).

## Installation

Python ≥ 3.10.

```bash
git clone https://github.com/dellebelle2500-git/ET_ARC.git
cd ET_ARC
python -m venv .venv && source .venv/bin/activate    # optional
pip install -r requirements.txt
```

### Fonts

Figures target Arial for journal compliance. If Arial is unavailable, the
scripts fall back to **Liberation Sans** (metrically identical to Arial) and
then DejaVu Sans. Liberation Sans ships with most Linux distributions
(`fonts-liberation`); on other systems install it, or edit the
`font.sans-serif` list at the top of each script. All scripts set
`pdf.fonttype = 42` so text is embedded as editable TrueType rather than
outlines.

## Usage

Run any figure script from anywhere; paths resolve relative to the repo root.

```bash
python figures/figure1.py      # -> output/Figure1.pdf , output/Figure1.png
python figures/figure2.py      # -> output/Figure2.pdf , output/Figure2.png
python figures/figure3.py      # -> output/Figure3.pdf , output/Figure3.png
python figures/figureS3.py     # -> output/FigureS3.pdf, output/FigureS3.png
```

Regenerate everything:

```bash
for f in figures/figure*.py; do python "$f"; done
```

## Figures

| Script | Figure | Panels |
|--------|--------|--------|
| `figure1.py` | Fig. 1 | **a** CRST spaghetti (paired *t*-test) · **b** QUEST spaghetti · **c** clinical-outcome PCA · **d** Ward dendrogram · **e** subgroup time-courses (mean ± SEM) · **f** baseline & 3-month change by subgroup (Kruskal–Wallis; pairwise Mann–Whitney U) |
| `figure2.py` | Fig. 2 | **a** CRST improvement by allele dosage (8 loci) · **b** dosage distribution across subgroups (G-test) · **c** volcano (183 SNPs, LBL) |
| `figure3.py` | Fig. 3 | **a** composite pharmacogenomic scores on PCA embedding · **b** two-axis pharmacogenomic landscape |
| `figureS3.py` | Fig. S3 | 4×4 grid: CRST-improvement anchor + 15 tiered loci overlaid on the PCA embedding |

### Conventions used throughout

- **Response subgroups** are ordered **SR → MR → SF** and drawn with a fixed
  marker for each (SR ● circle, MR ■ square, SF ◆ diamond). Where subgroup is
  encoded by colour (Fig. 1), the palette is SR blue, MR green, SF red.
- **Two functional axes**: *Metabolic Resilience* (positive-direction loci) and
  *Circuit Vulnerability* (negative-direction loci).
- **Locus tiers**: Tier 1 (cross-validated in both frameworks), Tier 2a
  (dosage-only), Tier 2b (subgroup-only).

Figures 4 and S4 (FDG-PET) are not included here; they depend on imaging data
generated by a collaborating site.
